🔧 数据库初始化

首次使用需要建表，请按以下步骤操作：

  1. 打开 Supabase 控制台 → 登录 → 点击项目 prgkwnzzuxirywyxkesb
  2. 左侧菜单点击 SQL Editor
  3. 点击 New Query（新建查询）
  4. 复制下面的 SQL，粘贴进去，点击 Run（运行）
CREATE TABLE IF NOT EXISTS approvals (
  id TEXT PRIMARY KEY,
  data JSONB NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

ALTER TABLE approvals ENABLE ROW LEVEL SECURITY;

DROP POLICY IF EXISTS "allow_all" ON approvals;
CREATE POLICY "allow_all" ON approvals
  FOR ALL USING (true) WITH CHECK (true);

运行成功后刷新页面即可使用。
